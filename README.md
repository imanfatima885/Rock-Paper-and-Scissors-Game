# Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. The game allows users to play against the computer, with scores updating automatically after each round.

## Features

* 🎮 Play Rock, Paper, or Scissors against the computer.
* 🤖 Random computer move generation.
* 📊 Live score tracking for both the user and the computer.
* 🎨 Responsive and clean user interface.
* 💬 Displays game result messages (Win, Lose, or Draw).

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
